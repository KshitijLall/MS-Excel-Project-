In this project , data analysis of Vrinda Store has been done. 
Here different types of problems are solved which mainly include that how to increase the sales.
Furthermore , a dashboard has been made using the Pivot Table.
VLOOKUP has also been used.  
These are the various questions that has been solved in this project:

Q.1 Compare the sales and orders using single chart.
Q.2 Which month got the highest sales and orders?
Q.3 Who purchased more - men or women in 2022?
Q.4 What are the different order status in 2022?
Q.5 List top 10 states contributing to the sales.
Q.6 Relation between age and gender based on number of orders.
Q.7 Which channel is contributing to maximum sales?
Q.8 Which is the highest selling category?
